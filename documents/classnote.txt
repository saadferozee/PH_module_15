NOTE_______________________________________________

    • For loop -->

        for loop structure -->

            for (variable declaration ; loop condition ; increment ) {
                statement 1;
                statement 2;
            }

                example -->
                    for ( let i = 1 ; i <= 10 ; i++ ) {
                        console.log(i)
                    }

    • Decremental For loop -->

        structure -->

            for (let i = 20 ; i >= 0 ; i--) {
                console.log(i)
            }

            note: we can use it in while loop as well.
